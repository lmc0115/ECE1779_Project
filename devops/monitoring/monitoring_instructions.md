# DigitalOcean Monitoring Setup

## 1. Enabled DO built-in monitoring:
- CPU usage
- Memory usage
- Disk I/O
- Network traffic

## 2. Created alerts:
- CPU > 70% for 5 min
- Memory > 80% for 5 min
- Droplet down / unreachable

## 3. Dashboard:
- Observed graphs for backend and db nodes
- Verified alerts send to UofT emails
